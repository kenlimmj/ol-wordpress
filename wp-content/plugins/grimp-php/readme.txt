== Description ==

Grimp - PHP allows you to use php code in pages and posts using the usual php markup (`<?php PHP_CODE ?>`).

== Installation ==

1. Upload `grimp-php.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php PHP_CODE ?>` in your post or page
